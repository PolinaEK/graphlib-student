# GraphLib

Небольшая учебная библиотека для работы с графами:
- создание графов
- алгоритм Дейкстры
- визуализация графов

## Описание проекта 

Учебная библиотека для работы с графами: создание, визуализация и алгоритмы на Python. Простая, понятная и удобная для студентов и преподавателей.
